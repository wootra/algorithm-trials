# Zigzag Conversion

given some sequence of characters, make it as zigzag pattern, and read lines from top to bottom without space.

ex>
given string

```txt
IamyourFather
```

convert it to zig zag pattern. when number of row is 4

```txt
I  r  r
a uF e
mo ah
y  t
```

and read lines without space

```txt
IrrauFemoahyt
```
